# Read-only movie API using Express

### Run locally

```bash
git clone git@github.com:kenziebottoms/nss-back-01-express-api.git
cd nss-back-01-express-api
npm install
npm run db:reset
npm start
```

## Usage

Each endpoint returns JSON.

| endpoint     | results                         |
| ------------ | ------------------------------- |
| `/directors` | Get all directors.              |
| `/movies`    | Get all movies.                 |
